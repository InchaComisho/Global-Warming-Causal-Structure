# 失われた10年・自然冷却シミュレーション

**概念的・反実仮想・因果シミュレーション（2015〜2035年）**

---

## このシミュレーションについて

本モジュールは、**概念的・因果的・インデックスベースの反実仮想モデル**です。

これは精密な気候予測ではありません。
℃、W/m²、GtC/yr などの物理単位で結果を出力するものではありません。
すべてのインデックスは0〜100に正規化された例示的な代理指標であり、公開されている方向性トレンドをもとに作成されています。

このシミュレーションの目的は、一つの因果的な問いを構造的に考察することです：

> **もし10年前、気候政策がCO₂削減やカーボンクレジットだけでなく、地球の自然冷却機能の回復と補完にも資源を向けていたなら、何が変わっていたか？**

---

## 「失われた10年」仮説

2015年から2025年にかけて、世界の気候政策は主に以下を中心に組織されていました：

- CO₂・温室効果ガスの排出削減目標
- カーボンクレジット・カーボンオフセット市場
- ネットゼロ誓約と排出量計算フレームワーク

これらの取り組みは現実のものであり重要です。しかし、本シミュレーションはそれらが構造的に不完全であったと提起します。なぜなら以下が十分に対処されなかったからです：

- 森林の蒸散と潜熱移送の崩壊
- 土壌微生物群集と土壌保水能の劣化
- 海洋生物生産性・植物プランクトン炭素固定の低下
- 大気・海洋の鉛直循環の弱体化
- 水の相変化冷却能（蒸発・凝縮・潜熱）の喪失

[地球温暖化の因果構造](https://github.com/InchaComisho/Global-Warming-Causal-Structure)で展開されている因果モデルにおいて、これらは周辺的な問題ではありません。地球が歴史的に気温と熱負荷を調整してきた主要メカニズムです。

「失われた10年」仮説とは：

> **もし2015年に世界の資源が、クーリングクレジットという金融メカニズムを通じてこれらの自然冷却機能の回復に振り向けられていたなら、2025年までの地球の熱負荷と自然冷却機能インデックスの軌跡は異なっていた可能性がある。**

### 期間

| 投影 | 2025〜2035年 |

### シナリオ

| ID | 名称 | 説明 |
|---|---|---|
| **A** | 観測値（歴史的） | CO₂中心・カーボンクレジット中心の対応。自然冷却機能の回復が不十分。 |
| **B** | CO₂削減強化 | 排出削減は強化されるが、自然冷却機能の回復は依然として弱い。 |
| **C** | 2015年から自然冷却機能回復開始 | 森林・土壌・水循環・蒸散・海洋支援を10年前から開始。 |
| **D** | 2015年からクーリングクレジット経路 | カーボンオフセット計算から測定可能な物理的冷却回復へと金融インセンティブを転換。 |

## モデル指標

| インデックス | 説明 |
|---|---|
| `forest_cooling_function` | 森林の蒸散と潜熱冷却能力 |
| `soil_microbe_function` | 土壌微生物活性と有機物循環 |
| `soil_water_retention` | 土壌保水能と地表温度緩衝能 |
| `evapotranspiration_function` | 植生全体の蒸散冷却能 |
| `ocean_biological_sink` | 海洋生物生産性と植物プランクトン炭素固定 |
| `vertical_circulation_function` | 大気・海洋の鉛直混合と熱再分配 |
| `water_phase_transition_cooling` | 蒸発・凝縮・潜熱移送による冷却 |
| `natural_cooling_function` | 上記7次元の複合インデックス |
| `carbon_sink_capacity` | 陸域・海洋の炭素吸収能力合計 |
| `heat_load_index` | 地球システムの蓄積熱エネルギー（高いほど悪化） |
| `cooling_credit_intervention` | クーリングクレジット政策・投資の強度 |
| `carbon_credit_strength` | カーボンクレジット・カーボンオフセット政策の強度 |

## 大きなシナリオ差の読み方

CO₂中心の経路と自然冷却機能回復の経路の間に大きな差が出ることは、精密な数値予測として読むべきではありません。これは、**介入している因果ノードが異なる**ことによる構造的な結果として読むべきです。

カーボンクレジット中心の経路が主に評価するのは、以下です。

- CO₂排出量
- 排出相殺
- 排出帳簿
- ネットゼロ主張

しかし、本シミュレーションの因果構造では、温暖化は大気中CO₂の増加だけでなく、熱を移動・吸収・緩衝・放散してきた自然システムの劣化によっても増幅されます。

- 森林蒸散
- 土壌保水力
- 土壌微生物機能
- 水の相変化冷却
- 海洋生物による炭素固定
- 大気・海洋の鉛直循環
- 都市・地表の熱蓄積

したがって、排出削減や排出相殺だけでは、失われた自然冷却機能は自動的には戻りません。逆に、因果の連鎖を特定し、弱体化したノードそのものへ資源を向ければ、回復は相互に強化される可能性があります。

```text
土壌回復
→ 保水力向上
→ 蒸散回復
→ 地表熱低下
→ 植生の生存率向上
→ 炭素固定力の回復
→ 熱負荷圧力のさらなる低下
```

このため、反実仮想シナリオでは大きな乖離が現れます。本モデルが示唆するのは、過去10年の問題が単なる努力不足ではなく、**不完全な因果診断によって気候資金の配分先がずれていた可能性**です。

公開上の結論は、次のように慎重に表現できます。

> 本シミュレーションは、正確な気温変化を予測するものではない。しかし、10年前からCO₂削減と並行して、失われた自然冷却機能の回復・補完に取り組んでいれば、熱負荷の蓄積と自然冷却機能低下の軌道は現在とは大きく異なっていた可能性がある。

## カーボンクレジットとクーリングクレジットの違い

| 次元 | カーボンクレジット | クーリングクレジット |
|---|---|---|
| **主要指標** | CO₂相殺・削減単位 | 回復・補完された物理的冷却機能 |
| **冷却機能** | 間接的・遅延あり | 直接的・測定可能 |
| **熱負荷効果** | CO₂蓄積を遅らせる | 蓄積熱負荷を直接低減 |
| **生態学的範囲** | 主に大気 | 陸域・海洋・土壌・水循環・大気 |
| **測定基盤** | CO₂計算（MRV） | 物理的冷却MRV（蒸散・土壌水分・海洋生産性） |
| **失われた10年との関連** | 単独では不十分 | 構造的ギャップを埋める |

定義の詳細：[クーリングクレジット定義](https://github.com/InchaComisho/Cooling-Credit-Definition)

## 重要な注意事項

- 全インデックス値は例示的な代理指標であり、直接測定された物理量ではありません。
- 森林冷却機能トレンドはFAO/FRAの世界森林消失推計と方向的に整合しますが、GtC/yrやkm²の精密な合計値ではありません。
- 炭素シンク弱体化はGlobal Carbon Budget陸域シンク異常レポートと方向的に整合します。
- 土壌微生物・蒸散・水相変化・自然冷却機能インデックスは代理インデックスであり、直接的な世界規模の測定合計値ではありません。
- このシミュレーションは摂氏単位での正確な気温変化を予測するものではありません。
- 目的は定量的予測ではなく、因果・構造的な推論です。
