# モデル前提条件 — 失われた10年・自然冷却シミュレーション

本文書はシミュレーションの前提条件、参照データ基盤、方法論的選択を詳述します。

---

## 1. インデックス正規化

すべてのモデル次元は **0〜100の正規化インデックス尺度** を使用します。

- **100** はその次元の機能が完全または最大健全な状態を表します。
- **0** はその機能が完全に喪失した状態を表します。
- 物理単位への変換は行いません。
- 値は例示的な代理指標であり、世界規模の直接測定値ではありません。

この正規化は意図的なものです。モデルの目的は定量的予測ではなく、因果・構造的推論です。

---

## 2. 時間軸

| 期間 | 状態 |
|---|---|
| 2015〜2025年 | 歴史的 / 観測方向（公開データトレンドに方向的に整合） |
| 2025〜2035年 | 各シナリオ下での投影（外挿、予測ではない） |

2025年の境界が本シミュレーションの現在年です。2025年以前の値は方向的な公開データに対して較正されています。2025年以降の値はシナリオベースの外挿です。

---

## 3. シナリオ定義

### シナリオA — 観測値 / 歴史的実績

過去10年の世界的な気候対応の軌跡と概ね整合します：

- CO₂中心の政策枠組み（パリ協定目標、NDC）
- カーボンクレジット・カーボンオフセット市場の成長
- 自然冷却機能の生態学的回復への投資が不十分
- 継続的な森林消失（FAO/FRA世界森林面積トレンドと整合）
- 継続的な土壌劣化（IPCC AR6土地利用報告と整合）
- 影響を受けた海域での海洋生産性の継続的な低下

冷却機能インデックスは以下の一般的な方向性コンセンサスから導いた低下率をたどります：
- FAO世界森林資源評価報告書
- Global Carbon Budget 2022〜2024の陸域シンク異常
- IPCC AR6 WG1・WG2の生態系劣化に関する知見

### シナリオB — CO₂削減強化

シナリオAと比べて排出削減は強化されるが、生態系の冷却機能回復は大幅には増加しない：

- 排出削減率はシナリオAより約50%速い
- カーボンクレジット市場はより速く成長
- 自然冷却機能回復はシナリオAとほぼ同水準
- 物理的な冷却喪失が継続するため、熱負荷の改善は限定的

### シナリオC — 2015年から自然冷却機能回復開始

2015年に世界の気候政策が自然冷却機能回復に焦点を当てた第二のトラックを追加したと仮定します：

- 大規模な森林回復プログラム開始（植林、再植林、森林破壊回避）
- 土壌健全性回復プログラム開始（有機物還元、菌根菌回復、農薬削減）
- 水循環回復プログラム開始（流域回復、湿地回復）
- 植生多様性回復による蒸散強化
- 都市冷却プログラム開始（緑の屋根、都市樹木、透水性舗装）
- 海洋生物支援（栄養塩流出削減、沿岸生態系回復）
- 地表温度変調による大気・海洋の鉛直循環支援

適用する回復係数：
- 2015年に低下を反転させ始め、凹型回復曲線（初期の速い回復、徐々に安定化）
- 2025年までに測定可能なプラスの変化を達成
- 2035年まで改善継続

### シナリオD — 2015年からクーリングクレジット経路

シナリオCを踏まえ、投資を体系的に転換する金融メカニズムを追加します：

- CO₂計算だけでなく**測定可能な物理的冷却**を報酬とするクーリングクレジット制度を確立
- 金融インセンティブを純粋なカーボンオフセット市場から冷却機能MRV（測定・報告・検証）へ転換
- すべての自然冷却次元の回復係数がシナリオCより約15〜20%強い
- クーリングクレジット介入インデックスが2025年までにほぼゼロから高水準へ上昇
- カーボンクレジット強度は維持されるが、一部がクーリングクレジットに置き換えられる
- 熱負荷低減軌跡が全シナリオ中最も良好

## 4. インデックス構成

### `natural_cooling_function`（複合）

各時点での以下7インデックスの**算術平均**として計算：

```
natural_cooling_function = mean(
    forest_cooling_function,
    soil_microbe_function,
    soil_water_retention,
    evapotranspiration_function,
    ocean_biological_sink,
    vertical_circulation_function,
    water_phase_transition_cooling
)
```

各次元の惑星冷却全体への相対的寄与は正確にはわからず、地域・季節によっても変化するため、一様加重を使用。一様加重は根拠のない精度を避けます。

### `heat_load_index`

蓄積変数としてモデル化。自然冷却機能が低く、CO₂強制力が高い場合に増加します。シナリオAではやや超線形トレンドをたどります。シナリオC・Dでは、自然冷却機能の回復が蓄積軌跡を遅らせ部分的に反転させます。

熱負荷インデックスはジュール単位の絶対的な海洋熱容量や℃単位の特定の気温異常を表しません。熱蓄積の方向性の構造的代理指標です。

### `carbon_sink_capacity`

陸域・海洋の炭素吸収能力の合計を表します。影響を受ける要因：

- 森林面積と状態（陸域シンク）
- 土壌有機物と微生物活性（陸域シンク）
- 植物プランクトンと海洋生物生産性（海洋生物ポンプ）

シナリオAでは、Global Carbon Budgetで観測された特定年・地域の陸域シンク弱体化と方向的に整合して低下します。

## 5. データ参照基盤

| 次元 | 参照基盤（方向のみ） |
|---|---|
| 森林冷却機能 | FAO世界森林資源評価（2015, 2020, 2025）；IPCC AR6 WG2 第2章 |
| 土壌微生物機能 | IPCC AR6 WG1 第5章；Lal（2004）土壌炭素文献レビュー方向 |
| 海洋生物学的シンク | Global Carbon Budget 2022〜2024；Friedlingstein et al. 陸域シンク異常データ方向 |
| 炭素シンク容量 | Global Carbon Budget 2022〜2024 |
| 鉛直循環 | IPCC AR6 WG1 第9章 海洋熱容量と成層方向 |
| 水相変化 | Trenberth et al. 全球水循環レビュー方向 |

すべての値は**方向的較正のみ**です。単一データセットの数値を精密に再現することは目標ではありません。

## 6. 補間法

2つの補間関数を使用します：

- **`_lerp(start, end, n)`**：単純線形補間。政策介入変数（クーリングクレジット、カーボンクレジット）に使用。
- **`_smooth_lerp(start, end, n, power)`**：調整可能な指数を持つベキ乗補間。生態系変数に使用。
  - 低下変数（power > 1.0）：わずかに加速する低下（ティッピングポイントリスクと整合）。
  - 回復変数（power < 1.0）：時間とともに鈍化する速い初期回復（生態系回復ダイナミクスと整合）。

確率的ノイズは適用しません。モデルは決定論的で例示的です。

## 7. モデルが主張しないこと

このモデルは以下を**主張しません**：

- クーリングクレジット単独で温暖化を止められたこと。
- 摂氏単位での精密な気温結果。
- GtC/yr単位での精密な炭素計算。
- 特定のインデックス値が正確な世界規模の測定値であること。
- 10年間のシナリオ乖離が唯一可能な解釈であること。
- シナリオDが重大な政治的・経済的課題なしに達成可能であること。

モデルは因果構造と気候対応経路の比較論理を探るための**概念的ツール**です。

## 8. 地球温暖化の因果構造との接続

このシミュレーションは以下で説明する因果モデルを操作化します：

> [地球温暖化の因果構造](https://github.com/InchaComisho/Global-Warming-Causal-Structure)
> [地球温暖化の因果構造 — GitHub Pages](https://inchacomisho.github.io/Global-Warming-Causal-Structure/)
> [NOTE記事：地球温暖化の原因と因果構造](https://note.com/inchacomusho/n/n5b2102ffc1c2)

そこでの因果仮説：

```text
産業文明が拡大する
→ 森林が伐採され、土壌が劣化し、海洋が汚染される
→ 蒸散、微生物活動、植物プランクトンが低下する
→ 陸域・海洋・大気にわたって自然冷却機能が弱体化する
→ 鉛直循環が弱まる
→ 地球システムで熱が停滞しやすくなる
→ 地球温暖化が加速する
```

このシミュレーションはその仮説を、時系列インデックスを持つ4つの比較可能なシナリオに変換し、「失われた10年」のコストを可視化します。

## 9. クーリングクレジット定義との接続

ここでモデル化したクーリングクレジットは以下の定義に対応します：

> [クーリングクレジット定義](https://github.com/InchaComisho/Cooling-Credit-Definition)

クーリングクレジットとは以下の行動に付与されるクレジット単位です：

- 物理的に熱負荷を低減する
- 自然冷却機能を回復する
- MRVで測定可能である
- 人類・文明・生態系のレジリエンスに貢献する

シミュレーションでは、`cooling_credit_intervention`インデックスがグローバル経済全体でのそのようなメカニズムの総合的な強度を表します。

## 10. 再現可能性

シミュレーションはPythonスクリプトから完全に再現可能です：

```bash
python lost_decade_natural_cooling_sim.py
```

すべてのパラメータと補間関数はスクリプトに明示的に定義されています。モデルの実行に外部データファイルは不要です。CSVとPNG出力は決定論的に生成されます。
